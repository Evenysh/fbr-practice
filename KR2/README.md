# KR2

В папке собраны практические работы по фронтенд и бэкенд разработке:

- `fbr_practice_7_8` — Node.js API: регистрация, авторизация, JWT, маршрут `/api/auth/me`, CRUD товаров, защищённые маршруты.
- `fbr_practice_9_10` — развитие предыдущего задания: refresh-токены, маршрут `/api/auth/refresh`, frontend на React.js, связанный с backend.
- `fbr_practice_11` — система ролей и прав доступа: guest, user, seller, admin; управление товарами и пользователями в приложении.
